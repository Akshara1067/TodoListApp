# Akshara Reddy - Django To-Do List Application

## Project Overview

This is a full-stack To-Do List web application developed using Django.  
The application allows users to manage daily tasks efficiently with authentication and CRUD functionalities.

---

## Features

- User Registration
- User Login and Logout
- Create Tasks
- Update Tasks
- Delete Tasks
- Mark Tasks as Complete
- Search Tasks
- User-specific task management

## Setup Instructions

### Step 1: Clone the Repository

```bash
git clone <repository-link>
```

### Step 2: Open Project Folder

```bash
cd todo_list
```

### Step 3: Create Virtual Environment

```bash
python -m venv venv
```

### Step 4: Activate Virtual Environment

For Windows:

```bash
venv\Scripts\activate
```

### Step 5: Install Required Packages

```bash
pip install -r requirements.txt
```

### Step 6: Run Migrations

```bash
python manage.py migrate
```

### Step 7: Start Development Server

```bash
python manage.py runserver
```
